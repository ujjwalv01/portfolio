# Ujjwal Verma - Personal Portfolio

A sleek, animated, and highly responsive personal portfolio website built to showcase my projects, skills, certifications, and competitive programming achievements.

## 🚀 Live Demo

*Update this section with your deployed website URL once live!*

## 🛠️ Tech Stack & Tools

- **Framework:** [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **UI Components:** [shadcn/ui](https://ui.shadcn.com/) & [Radix UI](https://www.radix-ui.com/)
- **Routing:** [React Router](https://reactrouter.com/)

## ✨ Key Features

- **Dynamic Hero Section:** Engaging entry with custom typing text algorithms and glowing interactive particle backgrounds.
- **Projects Showcase:** A clean gallery displaying major technical projects, tech stacks used, live URLs, and GitHub source codes.
- **Skills & Proficiencies:** Visual representation of frameworks, languages, and backend tools.
- **Competitive Programming Profile:** Highlights of live ratings and problem-solving badges across various CP platforms.
- **Certifications:** Documenting professional growth, courses, and technical accomplishments.
- **Responsive Navigation:** A seamlessly animated mobile & desktop navigation menu.

## 💻 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites
You need to have Node.js and npm installed.
- [Node.js](https://nodejs.org/en/download/)

### Installation & Run

1. Clone the repository:
   ```bash
   git clone https://github.com/ujjwalv01/portfolio.git
   ```
2. Navigate into the project directory:
   ```bash
   cd portfolio
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit the local port (usually `http://localhost:5173/`).
